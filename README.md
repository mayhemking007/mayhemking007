## Built

**[Searchify](https://github.com/mayhemking007/searchify)** — Built a search engine
from scratch to understand how one actually works. Custom crawler, hybrid retrieval
(BM25 + vector), RAG answer layer. Since I could only crawl so many links, I added
a "critical insight" layer that tells you what the answers are probably missing.
Constraint became a feature.
